# Ergodoxial keyboard

Design base on Ergodox keyboard, but with powerful ARM-based keyboard hub.
See [keymap source code](keymap.h)

#### RAISE key
* Tap `RAISE` + tap letter key: key is capitalized
* Double-tap `RAISE`: next word is capitalized: useful for entering `LONG_IDENTIFIERS`
* Hold `RAISE` + press keys: use navigation layer (see below) while `RAISE` is pressed.
* Slowly tap `RAISE`:  activate navigation layer. Red LED turns on to indicate Navigation mode. Tap `RAISE` again to de-activate.

#### Lower key
* Tap `LOWER` + tap letter key: enter one special character (from LOWER layer)
* Hold `LOWER` + type letter keys: enter several special characters

#### Del/FUNC key
* Tap `Del`: delete
* Hold `FUNC` + key: Function layer (`1`->`F1`, `2`->`F2`, etc.)
